# Calendar of events

# Description
This application allows: 
  * User registration
  * Log in to access the system
  * Logout
  * Add events
  * Edit events
  * Delete events 
  * View event listing

# Technologies
  * React
    * TypeScript  
  * CSS
    * Styled Components

# Dependencies
  * react-big-calendar
  * moment
  * react-toastify
  * react-hook-form
  * uuid
  * styled-components
