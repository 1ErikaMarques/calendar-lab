# Calendário de eventos

Esta aplicação web permite: 
  * Cadastro de usuário
  * Login para acesso ao sistema
  * Logout
  * Adicionar eventos
  * Editar eventos
  * Remover eventos 
  * Visualizar a listagem de eventos

# Tecnologias
  * React
    * TypeScript  
  * CSS:  
    * Styled Componets

# Dependências
  * react-big-calendar
  * moment
  * react-toastify
  * react-hook-form
  * uuid
  * styled-components
